### Gulp Training

This is a sample project to use the [gulp](https://gulpjs.com/) build tool.

#### Prerequesites:
- [NodeJS](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)

#### How to run
`npm build`
`gulp`
